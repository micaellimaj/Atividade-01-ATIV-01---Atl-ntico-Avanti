2. Explique o conceito de conjunto de treinamento, conjunto de validação e conjunto de teste em machine learning.

Um modelo de ML precisa ser treinado com um conjunto de dados e avaliado com outro, assim conseguimos melhor a capacidade preditiva do modelo com dados não vistos, pois ele não vai estar apenas decorando os dados e sim aprendendo com eles para gerar novos resultados. Então é feita a divisão em duas: uma maior de treino e outra menor de testes.
No treinamento o objetivo é que o modelo aprenda com o conjunto de dados de formar a generalizar bem para dados não vistos. Na validação o desempenho do modelo é avaliado visando ter uma estimativa de como ele está se saindo para dados não vistos. E no teste o objetivo é a avaliar a capacidade de generalização do modelo, só que desta vez com dados não visto, o que resulta em uma avaliação mais realista.

3: Explique como você lidaria com dados ausentes em um conjunto de dados de treinamento.

É uma boa prática sempre que for trabalhar com um novo conjunto de dados, verificar como eles estão estruturados. Então começaria importando para minha máquina, verificaria se há valores nulos no dataset, se sim, dependendo do caso, substituiria pela mediana ou media dos valores se for campo numérico, mas se for campo de texto, poderia substituir o valor por "desconhecido" ou o nome de maior ocorrência, nesse caso específico ou em outro precisaria entrar em contato com os tomadores de decisão para ver qual seria a melhor prática, pois dependendo do que eu substituir pode acabar impactando futuramente nas análises, gerando até inconsistência de resultados. Também poderia deletar as linhas que representasse esse problema, no entanto, se for um dataset, por exemplo, com informações importantes de clientes ou financeiro, isso pode acabar prejudicando, por isso uma boa comunicação entre a equipe é fundamental para evitar esse tipo de problema.

É uma técnica para avaliar a capacidade de predição do modelo treinado, como a acurácia que mostra a porcentagem de acertos do modelo treinado. Mas no caso da matriz de confusão temos um mapa de calor, quanto mais quente melhor, pois está tendo mais valores corretamente classificado.
